<h2 align="center">âš™ What you'll find inside</h2>
<ol>
  <h3><li>Outlier Detection (Unsupervised Learning)</li></h3>
  Explore how certain NBA players' statistical profiles stand apart. This notebook:
  <ul>
    <li>Performs comprehensive exploratory data analysis.</li>
    <li>Identifies outstanding performers based on offensive and defensive statistics.</li>
    <li>Uses clustering & statistical methods to spotlight these outliers using Isolation Forest and DBSCAN algorithms.</li>
    <li>The analysis identifies the top ten players by season, playoff, or All-Star performance</li>
  </ul>

  <h3><li>Game Outcome Prediction (Supervised Learning)</li></h3>
  Two notebooks form a complete predictive pipeline:
  <ol>
    <li><b>Game Outcome Prediction Part 1:</b><br>
      Uses Ensemble Learning with Linear Regression, Decision Tree, Random Forest, Gradient Boosting, XGBoost Regressor, and Support Vector Regressor algorithms.</li>
    <li><b>Game Outcome Prediction Part 2:</b><br>
    Uses K-Means Clustering for clustering analysis:
    <ul>
      <li><b>Team Performance Clustering:</b> captures a complete view of team capabilities, considering both offense and defense metrics to analyze detailed performance patterns.</li>
      <li><b>Point Difference Clustering:</b> uses the specific point_differential metric to create a straightforward winning vs. losing team distinction, which is often predictive of game outcomes when scoring is a primary performance indicator.</li>
    </ul>
    </li>
  </ol>
